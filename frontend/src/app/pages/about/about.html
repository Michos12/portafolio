About me

Full Stack Developer · Angular + Python

I'm a Full Stack Developer who enjoys taking products from idea to production: clean, accessible interfaces on the front, and well-structured, secure APIs on the back.

I care about the details that make software trustworthy — input validation, authentication, rate limiting, sensible error handling and fast load times. I build with a decoupled mindset so each layer can scale and deploy independently.

This portfolio is a working example: an Angular SPA on the front, a FastAPI service on the back, and a PostgreSQL database — each hosted on its own platform.
